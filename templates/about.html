About the Project

This web application is the final project for the DAB111 course (Python). It involves the entire process of collecting, cleaning, and displaying data using web scraping, data processing, and Flask web development.

Data Collection

We utilized BeautifulSoup (bs4) to scrape data from a website, extracting relevant information for a race event.

Data Cleaning

The collected data underwent a cleaning process to handle missing values and ensure accuracy and consistency.

Data Storage

The cleaned data was stored using SQLite3 to create a database, allowing for efficient data retrieval and management.

Webpage Development

Flask, a Python web framework, was employed to develop this webpage. The framework facilitates the integration of data and presentation, creating a seamless and interactive user experience.

Project Acknowledgment

This project serves as the crown of skills and knowledge acquired during the DAB111 course, showcasing the ability to apply data collection, cleaning, and web development concepts in a real-world scenario.
